Implement quick sort (30 points)

Using the pseudocode in the textbook as a guide, implement in Java (yes, this time you must use Java) the randomized quick sort algorithm as a method. You will also, of course, need to implement the methods it relies on.

Write a main method that:

  1. generates an array of length 100,000 with integer values randomly generated between -106 and 106
  2. records the start time in milliseconds
  3. calls quick sort
  4. records the end time and computes and prints the time the sort took to complete
  5. records the start time in milliseconds
  6. calls quick sort a second time on the same array, which is now sorted
  7. records the end time and computes and prints the time that call to the sort took to complete
You should see two similar times. If the second time is larger, you're probably not implementing the randomized partition algorithm correctly
